# Patterns

Reusable patterns that appear in this codebase. Record them only after you see the pattern applied 3+ times — premature patterns are worse than none.

Format for each entry:

- **Added:** YYYY-MM-DD
- **Where used:** 2-3 representative file paths
- **Shape:** short description of the pattern
- **When to use:** the trigger
- **When NOT to use:** anti-pattern signals
